FBReader® Book Network is a cloud storage for eBooks. It is based on the Google Drive™

Calibre is a popular eBook manager for your local computer

Implemented by November 27, 2014:

  • Download books from the Book Network and add them immediately to the Calibre Library

To be implemented:

  • Save cookies between sessions (do not login to the network every time)
  • Search books on the Book Network

  • (?) Automatically synchronise local library with the Book Network
  • Update books in the Book Network from Calibre
